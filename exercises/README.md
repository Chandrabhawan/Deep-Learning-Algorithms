# Exercises

These are per-module exercise notebooks designed to reinforce the concepts covered in each corresponding module of the Deep Learning Algorithms course.

## Structure

Each notebook contains **5-10 exercises** with a mix of:

- **Concept checks** -- multiple choice or short answer questions to test understanding
- **Implementation tasks** -- hands-on coding exercises where you build components from scratch
- **Debugging tasks** -- broken code snippets where you must identify and fix errors

## Solutions

Solutions for every exercise are included at the bottom of each notebook under **collapsible headings**. Click the triangle or summary text to expand a solution. Try to work through each exercise on your own before checking the answer.

## Recommended Workflow

1. Complete the corresponding module notebook in `../notebooks/` first.
2. Open the matching exercise notebook here (e.g., after finishing `DL100`, work through `DL100_exercises.ipynb`).
3. Attempt each exercise without looking at the solutions.
4. Check your answers against the provided solutions.
5. Revisit the module material for any topics where you struggled.
